# 泰山職訓前端班期中作業
請學員製作⼀個查詢公開資料的 LINE 機器⼈，並部署到 [render.com](https://render.com/) 雲端伺服器  
可使⽤ Google API、[政府開放資料](https://data.gov.tw/)等資料來源  

說明:
- 主要有三個功能，打招呼、日常回覆及回地址回傳。
- 剛開始還能回傳地址，後來修改程式邏輯後就壞掉了。
- 改兩三個禮拜改不好，實在不行了。
